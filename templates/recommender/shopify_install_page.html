The recommendation system works based on product names and tags. Add these tags to your Shopify products to improve accuracy:

Skin types: dry, normal, oily Skin concerns: darkcircle, skinredness, melasma, vascular, wrinkle Acne: blackhead, cystic, folliculitis, keloid, milium, papular, purulent, acne_scars, acne, pimple, spot You are not required to add all tags — just add the ones relevant to each product.
