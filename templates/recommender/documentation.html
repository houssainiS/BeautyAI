Beautyxia Documentation

Complete guide to setting up and using your Shopify app

1App Installation

  • After installing the app, you are redirected to the "App Installed Successfully" page.
  • On this page, you will see two buttons:
    • Create Page & Navigation Link: Automatically creates a dedicated page in your store and adds a nav bar link for the app.
    • Open Theme Editor (Deep Link): Opens the Shopify theme editor directly on the page created for the app.

2App Features

Face Analysis
Customers can upload a photo, which is analyzed temporarily by the backend. The system uses AI to detect skin type, skin conditions (like redness, dark circles, melasma), and acne-related issues. No photos are stored permanently; only temporary analysis is performed to generate personalized beauty tips.
Product Recommendations
The recommendation system suggests products based on the product names and their tags. To make it work effectively, merchants should add relevant tags to products. Skin types: 'dry', 'normal', 'oily'
Skin concerns: 'darkcircle', 'skinredness', 'melasma', 'vascular', 'wrinkle'
Acne & related issues: 'blackhead', 'cystic', 'folliculitis', 'keloid', 'milium', 'papular', 'purulent', 'acne_scars', 'acne', 'pimple', 'spot'
Merchants are not required to add all tags—only the ones relevant for each product.
Navigation & Theme
Merchants can create and edit pages with our app using the provided buttons.

3Security & Compliance

  • App uses HTTPS, Django sessions, CSRF protection, and limited access to logs.
  • Complies with GDPR, including explicit consent for photo analysis and handling of IP/email info.

Security Features: All data transfers are encrypted, sessions are handled securely, and access to logs is strictly limited.
